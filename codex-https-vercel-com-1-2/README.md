# 케어링크

수원시, 용인시, 성남시를 대상으로 하는 출장 웰니스 마사지 중개 플랫폼 초안입니다.

## 핵심 기능

- 한국어 UI
- 시 → 행정구 → 행정동 3단계 지역 선택
- 수원시, 용인시, 성남시 지역 데이터 포함
- 파트너 추천 카드
- 예약 상담 요청 UI
- 합법적인 웰니스 방문 케어 운영 안내

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 열면 됩니다.

## Vercel 배포

Vercel 계정에 로그인한 뒤 프로젝트 폴더에서 아래 명령을 실행합니다.

```bash
vercel deploy
```

GitHub 저장소와 Vercel을 연결했다면, 이 프로젝트를 커밋하고 푸시하면 자동으로 배포됩니다.
